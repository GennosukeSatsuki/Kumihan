# Text to Word Converter (Kumihan)

テキストファイルを Word (.docx) 形式に変換するデスクトップアプリケーションです。
Windows と Mac の両方に対応するため、**Tauri** を使用して開発します。

## 概要
ユーザーが選択したテキストファイル（.txt）を読み込み、Microsoft Word（.docx）形式として出力します。
特に日本語の創作活動（小説など）に特化した、詳細なレイアウト設定が可能です。

## 技術スタック
- **Framework**: [Tauri](https://tauri.app/) (Version 2)
- **Frontend**: HTML / CSS / JavaScript (Vanilla)
  - UI はモダンでプレミアムなデザイン（ダークモード対応、アニメーション、グラスモーフィズムなど）を目指します。
- **Backend**: Rust
  - ファイル操作および `.docx` の生成ロジックを担当します。
  - `docx-rs` クレートを使用して、高度な書式設定を実現します。

## 主要機能
- [ ] テキストファイルのドラッグ＆ドロップ入力
- [ ] 複数ファイルのバッチ変換
- [ ] **縦書き / 横書きの切り替え**
- [ ] **ノンブル（ページ番号）の有無設定**
- [ ] **詳細な書式・レイアウト設定**
    - 1行の文字数、1ページの行数（例：40文字×40行）
    - フォントサイズ、フォントファミリー
    - 指定した文字数・行数に基づいた**余白の自動計算**
- [ ] 出力先フォルダの選択
- [ ] 変換後の自動オープン機能（オプション）

## 開発ロードマップ
1.  **環境構築**: Tauri プロジェクトの初期化 (Done)
2.  **UI設計**: 美しく直感的なインターフェースの構築
3.  **バックエンド実装**: Rust によるテキスト解析と Word ファイル生成ロジックの構築
4.  **連携**: フロントエンドからの呼び出し（Invoke）と進捗表示の実装
5.  **ビルド・テスト**: Windows/Mac 両環境での動作確認

## 実行方法
```bash
npm install
npm run tauri dev
```
